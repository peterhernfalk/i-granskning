Ett alternativ till att ladda ner hela domänen är att ladda ner IS och TKB via länkar
som scriptet kan bygga upp baserat på domännamn och versionsnummer.

Länkexempel:
https://bitbucket.org/rivta-domains/riv.clinicalprocess.healthcond.certificate/src/4.0.5/docs/IS_clinicalprocess_healthcond_certificate.docx
https://bitbucket.org/rivta-domains/riv.clinicalprocess.healthcond.certificate/src/4.0.5/docs/TKB_clinicalprocess_healthcond_certificate.docx

Undantag: TK-namn ingår i dokumentnamnet på IS men inte på TKB
https://bitbucket.org/rivta-domains/riv.clinicalprocess.healthcond.actoutcome/src/4.0.1/docs/IS_clinicalprocess_healthcond_actoutcome_getLaboratoryOrderOutcome.docx
https://bitbucket.org/rivta-domains/riv.clinicalprocess.healthcond.actoutcome/src/4.0.1/docs/TKB_clinicalprocess_healthcond_actoutcome.docx
